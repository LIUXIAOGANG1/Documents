百度钱包接入资料介绍：
1.demo.rar里包含三个语言版本的示例（PHP/.NET/JAVA），里面有测试商户号，可以直接用；
2.建议商户先运行一次demo看看效果 之后再开始接入；
3.商户在开发的过程中尽量使用测试商户号，开发完成后再换成自己的商户号；
4.如果商户要进行压力测试，请自觉打桩，不要对百度钱包的收银台做压力测试，也没有必要，如果有活动等需要负载量过大，需提前联系百度钱包，这边会合理安排，满足商户的需要；
5.接入的过程中有问题可以联系技术支持协助解决。
6.商户上线前要联系百度钱包的技术支持，确定对接的没有问题后再上线，感谢配合！
